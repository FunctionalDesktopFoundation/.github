# The Functional Desktop Foundation of Philadelphia, Pennsylvania

Hey! We're the FDF. An independent software design studio in Philadelphia, Pennsylvania with international collaborators.   
We build software that makes (a select few) developers love us, and some UX designers as well.   

- [qmX](https://qmx.functionaldesk.top) + bridge is our flagship meta-framework and language for QML that provides React and Python-like abstraction and interactivity.
- [FDFlib](https://github.com/FunctionalDesktopFoundation/fdflib) provides our GUI primitives, and [stdlib](https://github.com/FunctionalDesktopFoundation/stdlib) provides its lower-level interactions.
   
Please support us on [Liberapay](https://liberapay.com/fdf) to help fund us and keep us going!
